# Universal Video Downloader
A simple and professional video downloader for Termux users.

## Features
- Download from YouTube, TikTok, and Facebook.
- Multiple quality options (1080p, 720p, 360p).
- MP3 conversion support.
- Clean UI with Progress Bar.

## How to Install
1. pkg install git python ffmpeg -y
2. pip install yt-dlp
3. git clone https://github.com/Hantkyaw/Universal-Video-Downloader.git
4. cd Universal-Video-Downloader
5. python k2dl.py
